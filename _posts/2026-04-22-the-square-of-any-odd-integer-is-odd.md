## Theorem

The square of any odd integer is odd.

---

## Definitions

An integer is **odd** if it can be written in the form:

[
2k + 1
]

for some integer ( k ).

---

## Proof

Let ( n ) be any odd integer.

By definition, there exists an integer ( k ) such that:

[
n = 2k + 1
]

Now square both sides:

[
n^2 = (2k + 1)^2
]

Expand:

[
n^2 = 4k^2 + 4k + 1
]

Factor the even part:

[
n^2 = 2(2k^2 + 2k) + 1
]

Since ( 2k^2 + 2k ) is an integer, this shows that ( n^2 ) has the form:

[
2m + 1
]

for some integer ( m ). Therefore, ( n^2 ) is odd.

---

## Conclusion

> If ( n ) is odd, then ( n^2 ) is odd.

---

## Intuition

Odd numbers are “one more than an even number.”

When you square an odd number, almost everything in the expansion is even, except for the final ( +1 ), so the result stays odd.
